# 🦑 Squid Game Problem Solving Assessment

Welcome to my submission for the **Squid Game Problem Solving Assessment**. This repository contains TypeScript implementations for 10 challenging algorithmic problems.

## 🚀 Overview

This project was built to demonstrate problem-solving skills using **TypeScript** and **Node.js**. Each solution is optimized for readability and performance, following modern coding standards.

## 🛠️ Tech Stack

- **Language:** [TypeScript](https://www.typescriptlang.org/) 📘
- **Runtime:** [Node.js](https://nodejs.org/) 🟢
- **Testing:** Custom test runner built directly into the solution file. 🧪

## 📂 Problems Solved

I have implemented the following 10 problems from the assessment:

1.  **String Reversal** 🔄 - Efficiently reversing strings.
2.  **FizzBuzz** 🐝 - Standard logic for multiples of 3 and 5.
3.  **Find Largest Number** 📈 - Single-pass maximum value detection.
4.  **Palindrome Check** ⬅️➡️ - Case-insensitive word symmetry verification.
5.  **Array Summation** ➕ - Using functional `reduce` for total values.
6.  **Factorial Calculation** ❗ - Iterative approach to avoid stack overflow.
7.  **Even Number Filtering** 🔢 - Extracting even integers from arrays.
8.  **Fibonacci Sequence** 🧬 - Generating custom range Fibonacci numbers.
9.  **Prime Number Check** 💎 - Optimized primality testing logic.
10. **Duplicate Removal** 🧹 - Using `Set` for clean, ordered unique arrays.

## 🏃 How to Run Locally

To verify the solutions on your local machine, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/shakil-ahmed-billal/problem-solving.git
    cd problem-solving
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the solutions:**
    ```bash
    npx ts-node solutions.ts
    ```
